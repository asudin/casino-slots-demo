# Casino Slots Demo
A demo project made to showcase a casino management game. Player can accumulate ordinary and premium currency, use it to buy casino slots, which have progression for passive income. There is shop, where player can convert his common currency to premium. Additionally, there are 5 different slots, where player can play and earn actively income.
